A Tetris game based on PyGame

Game Control:
   left arrow  : move left
   right arrow : move right
   down arrow  : soft drop
   space       : harddrop
   up arrow    : spin clockwise
   Z           : spin counter-clockwise
   C           : hold
   F2          : restart

Current status:

   basic GUI (quite bad)
   SRS (super rotation system)
